Build My App

This project connects users that have great ideas for mobile or web apps with the developers capable of turning those ideas into reality. Build My App was a 4 week long group project with 5 team members, built with React, Node, Express, PostgreSQL.

Project Overview

  • Implemented authentication using Auth0
  • Built API routes for messaging between users using Nodemailer.
  • Implemented custom middleware to decrypt JSON web tokens and authenticate protected API routes.
  • Created an interactive training module to help users understand how the app works.
  • Created multiple frontend components used for rendering users and profile pages.

Air Fitness

Providing trainers and clients the freedom to find each other and connect outside of a gym. Air Fitness gives trainers the ability to keep track of attendance and for clients to find new classes. Air Fitness was built with HTML, CSS, JS, React, Node, Express and PostgreSQL. It was built in 4 days with a 3 member team.

Project Overview

  • Built a single page application using React.
  • Implemented routing using react-router and handled AJAX requests using axios.
  • Styled using CSS

Droom


The idea behind Droom is to bring employers and job seekers together in a way where mutual interest is assured. Employers create job postings and approve or disapprove candidates and vice versa for job seekers. When a mutual match has been made, both parties are notified. Droom was built with HTML, CSS, JS, React, Node, Express and SQLite. This was a 4 day project with 3 team members.

Project Overview

  • Created a fully responsive marketing site for Droom using HTML, CSS and Javascript.
  • Built the layout using Flexbox
  • Implemented styles using LESS for preprocessing
