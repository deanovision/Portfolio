Projects

Here are some fun projects that I have worked on recently.

Build My App

This project connects users that have great ideas for mobile or web apps with the developers capable of turning those ideas into reality.

Project Overview

  • 4 week long group project with 5 team members
  • Built with React, Node, Express, PostgreSQL
  • Some of my contributions include custom route authentication middleware for the backend API, frontend components used for rendering users and profile pages as well as an interactive training module.

Air Fitness

Providing trainers and clients the freedom to find each other and connect outside of a gym. Air Fitness gives trainers the ability to keep track of attendance and for clients to find new classes.

Project Overview

  • 4 day project with 3 team members
  • Built with HTML, CSS, JS, React, Node, Express, PostgreSQL
  • I created the frontend as a single page application using React and styled with CSS. Some key APIs include axios and react-router.

Droom

The idea behind Droom is to bring employers and job seekers together in a way where mutual interest is assured. Employers create job postings and approve or disapprove candidates and vice versa for job seekers. When a mutual match has been made both parties are notified.

Project Overview

  • 4 day project with 3 team members
  • Built with HTML, CSS, JS, React, Node, Express, SQLite
  • I contributed by building a fully responsive marketing site for Droom using HTML, LESS/CSS and Javascript.
